Mentoring Groups Assessment

  • You must complete this assessment before joining our mentoring groups. The assessment will give us a chance to get to know you, your skill level and understand what your hoping to accomplish.
  • This assessment will require you to do the following:
    1. Setup a development environment of your choice
    2. Create a GitHub account if you don't already have one
    3. Create a GitHub page
    4. Create a basic HTML5 page and style it with basic CSS
    5. Use Google Chrome's Developer tools
    6. Use basic command line to navigate and manage your folders
