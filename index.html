Mentoring Groups Assessment

  • You must complete this assessment before joining our mentoring groups. The assessment will give us a chance to get to know you, your skill level and understand what your hoping to accomplish.
  • Our assessment will require you to do the following:
    1. Setup a development environment of your choice
    2. Create a GitHub account if you don't already have one
    3. Create a GitHub page
    4. Create 3 basic HTML5 pages and style them with basic CSS
    5. Use Google Chrome's Developer tools
    6. Use basic command line to navigate and manage your folders

  1. Development Environment

    • Browser: Google Chrome
    • Use whatever IDE you are most comfortable with. Sublime Text is a good free choice with a lot of on line support. You can take a look at this tutorial if interested.

  2. Git

    What we will be using for source and version control. If you are new to Git and want to learn more this a great guide: git- the simple guide

    • Create a GitHub account if you don't already have one
    • Setup Git and Generate SSH keys
    • Create a GitHub page

  3. HTML5 pages

    Your pages should contain the following tags: section, p, h1, h2, ul, li, img . They should also be stylized using CSS3: try adding different colors to each one of your sections, a border around your image and play around with some hover effects. Make sure you are comfortable with using Google Chrome developer tools .

    Commit all these pages to your GitHub page repository

    • Page 1: About me - could include a photo, short bio and a list of things you like
    • Page 2: What you want to learn and why you would like to be in a mentor group
    • Page 3: Projects you have worked on, working on or would like to have time to work on

  4. Basic Command Line (CLI)

    If you are using a Mac you can access command line through Terminal
    If you are using Windows you can access it through PowerShell

    Commands you should be familiar with

    1. ls
    2. mkdir
    3. cd
    4. rm
    5. mv
    6. cp
    7. more and less
    8. pwd
    9. find
    10. cat
